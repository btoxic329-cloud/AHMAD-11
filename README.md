Android Login Application

 Project Overview

This is an **Android Login Application** developed as an **individual semester project** using **Android Studio**.
The project demonstrates core Android concepts including **UI design, activity navigation, user input handling, and Gradle-based project setup**.

 Project Objectives

* Understand Android UI components
* Implement login screen functionality
* Practice activity navigation using intents
* Learn Android project structure and Gradle build system
* Prepare a complete, documented Android project for semester evaluation

---

Technologies Used

* **Android Studio**
* **Java / Kotlin** (as used in project)
* **XML** (UI Layouts)
* **Gradle** (Build System)

---

 Features

* Simple and clean login interface
* Username and password input fields
* Button-based interaction
* Navigation between activities
* Proper Android project structure
* Gradle synced and build-ready project

---

How to Run the Project

1. Open **Android Studio**
2. Click **File → Open**
3. Select the project folder
4. Wait for **Gradle Sync**
5. Connect a physical device or start an emulator
6. Click **Run ▶**
